# Express token based authentication

Example of token based authentication with Node.js, Express.js, MongoDB and Mongoose.

## RESTful API endpoints

### POST `/api/users`

Create a new user.

+ Method: `POST`
+ URL: `/api/users`
+ Body:

```js
{
  "username": "art",
  "password": "secret"
}
```

### POST `/api/users/authenticate`

Authenticate user.

+ Method: `POST`
+ URL: `/api/users/authenticate`
+ Body:

```js
{
  "username": "art",
  "password": "secret"
}
```
